## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Utrada/Flutter_All_3MCA_Code.git
```

### 2️⃣ Navigate to the Project

```bash
cd Flutter_All_3MCA_Code
```

### 3️⃣ Install Dependencies

```bash
flutter pub get
```

### 4️⃣ Run the Project

```bash
flutter run
```
